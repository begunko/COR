## 🎯 О проекте

**COR Engine** — это браузерный 3D-движок, созданный для командной работы над игровыми мирами в реальном времени.

В отличие от Unity и Unreal, COR работает прямо в браузере. Ничего не нужно скачивать — открыл ссылку и попал в редактор.

### Ключевая фишка
**До 1000 человек** могут одновременно редактировать один и тот же мир. Как Google Docs, но для 3D-миров.

## 🚀 Быстрый старт

### Требования
- Python 3.14+
- 16+ GB RAM (для AI-генерации)
- Ollama (для AI)

### Установка

```bash
# 1. Клонируем
git clone https://github.com/yourusername/cor-engine.git
cd cor-engine

# 2. Создаём виртуальное окружение
python -m venv venv
source venv/bin/activate  # Linux/Mac
venv\Scripts\activate     # Windows

# 3. Устанавливаем зависимости
pip install -r requirements.txt

# 4. Применяем миграции
python manage.py migrate

# 5. Создаём суперпользователя
python manage.py createsuperuser

# 6. Запускаем сервер
python manage.py rundev
